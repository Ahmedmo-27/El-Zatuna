## üìö Overview

El Zatuna is a comprehensive, enterprise-grade Learning Management System (LMS) designed to deliver exceptional online learning experiences. The platform empowers instructors, students, and organizations to create, manage, and deliver educational content at scale.

### üéØ Key Highlights

- **Multi-Format Courses**: Support for video courses, live classes, and text-based learning
- **Real-Time Learning**: Integrated video conferencing for live classes
- **Assessment & Certification**: Comprehensive quiz system with automatic certificate generation
- **Multi-Language Support**: Available in English, Arabic, Spanish, and more
- **Secure API**: RESTful API with modern authentication and security features
- **Payment Integration**: Multiple payment gateways for global transactions
- **Advanced Features**: Forums, assignments, live chat, student progress tracking, and analytics

## üõ†Ô∏è Technology Stack

### Backend
- **Framework**: Laravel 10.48
- **Language**: PHP 8.2+
- **Database**: MySQL 8.0
- **Authentication**: JWT + Laravel Sanctum (dual-token system)
- **API**: RESTful with versioning (v1)
- **Queue System**: Redis-backed job processing
- **File Storage**: AWS S3, Cloudflare R2, Local

### Frontend
- **Framework**: Blade Templates + Bootstrap 4
- **JavaScript**: jQuery, Vanilla JS
- **UI Components**: Design 1 (Modern responsive design)
- **Icons**: IconSax (Blade Components)
- **Charts**: Chart.js
- **Real-Time**: Agora SDK, Twilio
